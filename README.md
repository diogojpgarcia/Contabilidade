# 💰 Finanças Familiares v3.0 - Professional Edition

App de gestão de finanças familiares com sistema de segurança avançado, 30+ categorias, e análise profissional.

## 🆕 O Que Há de Novo na v3.0

### 🔐 FASE 1: Sistema de Segurança Híbrido

#### Proteção Completa Contra Perda de Dados
- ✅ **Pergunta Secreta** - Recuperação de PIN se esqueceres
- ✅ **Código de Recuperação** - Código de emergência (formato: XX-XXXX-XXXX)
- ✅ **Backup Automático** - Backup local encriptado a cada sessão
- ✅ **Exportar/Importar** - Ficheiro .encrypted para backup manual
- ✅ **Google Drive (Opcional)** - Backup automático na cloud (encriptado)
- ✅ **Encriptação AES-256** - Dados sempre protegidos

#### Como Funciona
1. **Setup Inicial** (primeira vez):
   - Crias PIN (4 ou 6 dígitos)
   - Escolhes pergunta secreta + resposta
   - Recebes código de recuperação (GUARDA EM PAPEL!)
   - Opcionalmente conectas Google Drive

2. **Se Esqueceres o PIN**:
   - Opção A: Respondes à pergunta secreta → Novo PIN
   - Opção B: Usas código de recuperação → Restaura dados

3. **Se Perderes TUDO** (limpar browser):
   - Importas ficheiro de backup + código de recuperação
   - Ou restauras do Google Drive

## 🔐 Setup de Segurança (Primeira Vez)

1. **Escolhe utilizador** (Diogo/Leila)
2. **Cria PIN** (4 ou 6 dígitos)
3. **Escolhe pergunta secreta**
   - Ex: "Cidade onde nasceste?"
   - Define resposta
4. **GUARDA o código de recuperação!**
   - Formato: XX-XXXX-XXXX
   - Anota em papel ou password manager
   - **SEM ESTE CÓDIGO NÃO HÁ RECUPERAÇÃO!**
5. **(Opcional)** Conecta Google Drive para backup automático

## 🆘 Recuperação de PIN

### Método 1: Pergunta Secreta
1. Ecrã de login → "Esqueci o PIN"
2. Escolhe "Pergunta de Segurança"
3. Responde à pergunta
4. Cria novo PIN

### Método 2: Código de Recuperação
1. Ecrã de login → "Esqueci o PIN"
2. Escolhe "Código de Recuperação"
3. Introduz código (XX-XXXX-XXXX)
4. Ou escolhe ficheiro de backup
5. Cria novo PIN

## 💾 Backup & Restauro

### Backup Manual
1. Settings → "Backup & Recuperação"
2. "Exportar para Ficheiro"
3. Download ficheiro `.encrypted`
4. Guarda em local seguro

### Backup Automático
- Criado automaticamente em cada sessão
- Guardado no browser (localStorage)
- Encriptado com código de recuperação

### Restaurar
1. Settings → "Importar de Ficheiro"
2. Escolhe ficheiro `.encrypted`
3. Introduz código de recuperação
4. Dados restaurados!

## 🔒 Segurança

### Encriptação
- **PINs**: Hash SHA-256 (nunca em texto simples)
- **Backups**: AES-256 com código de recuperação como chave
- **Dados**: End-to-end encryption (nem Google vê)

### Armazenamento
- **LocalStorage**: Dados do utilizador + transações
- **SessionStorage**: Sessão ativa
- **Backups**: Encriptados em localStorage ou Google Drive

## 🗄️ LocalStorage Keys

```
user_diogo              - Dados + PIN hash do Diogo
user_leila              - Dados + PIN hash da Leila
transactions_diogo      - Transações do Diogo
transactions_leila      - Transações da Leila
recovery_diogo          - Pergunta + código do Diogo
recovery_leila          - Pergunta + código da Leila
auto_backups            - Backups automáticos locais
drive_backups           - Backups Google Drive (fallback)
```
